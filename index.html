What is the difference between "switch" and "if/else"? When would you use one versus the other?

Switch statement is a way of comparing values. It is indicated by switch() case [value]:. No variables can be declared within a switch statement. By doing this, the computer compares the value of the case compared to the other items in a list. Switch is used when you have a finite list of correct answers, like months. If you have an input that's a month you write a switch and each month would be a case. "If/else" is a more commonly used comparison method for more open-ended cases. The "if" statement defines the condition using a Boolean statement and indicates what the program should do if those have been met. If those have not been met exactly, the "else" statement and action will run. "Else" can only be used immediately after an "if statement."

What is "void" in a method declaration? How does this differ from declaring a method with a return type?

A method must be called in order to be used -- "void" means no value will be returned, and a statement means a value will be returned.

The syntax of a method is: public static int methodName(int a, int b) {

// body

}

  • public static = modifier
  • int = return type
  • methodName = name of the method
  • a, b = formal parameters
  • int a, int b = list of parameters

    • "Void" means that any type can be declared, such as int, String, boolean, or double. If parameters are declared, it means the program needs that information to produce the desired output. Parameters can be passed to a void method. The difference is that a void method can do an operation on something that's in memory. Something with a return value can be passed to another method directly without using a class variable.
